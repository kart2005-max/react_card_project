# React Card Project

This project is a simple React application that demonstrates the creation of reusable card components using props and state.

## 🚀 Features
- Reusable Card Component
- Dynamic titles passed via props
- Like / Unlike toggle functionality using useState
- Clean and modular component structure
- Interactive UI

## 🛠️ Technologies Used
- React JS
- JavaScript (ES6+)
- HTML5
- CSS3
- Vite (for fast development build)

## 📂 Project Structure
- `src/components` → Contains Card component
- `src/App.jsx` → Main app logic
- `public/` → Static files
